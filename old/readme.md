
窗棂是我设想的 Linux 桌面, 主体如同 Compiz 桌面展览  
所有桌面都是 n*n 平铺的小桌面, 每个小桌面一个应用占满  
再引入动态桌面, 按需增减桌面行列, 保持紧凑  
再引入桌面嵌套设计, 一个小桌面可以嵌套更多桌面  
再引入共享功能, 让这个桌面访问和运行他人电脑的桌面  

网页展示的部分属于美工方面设想
